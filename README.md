# Sistem Peminjaman Ruangan (Frontend)

Aplikasi web frontend untuk sistem peminjaman ruangan kampus, dibangun menggunakan React, TypeScript, dan Tailwind CSS.

## 🚀 Fitur
- **User Dashboard:** Melihat daftar ruangan, status peminjaman, dan riwayat.
- **Admin Dashboard:** Mengelola ruangan (CRUD) dan persetujuan peminjaman (Approve/Reject).
- **Booking System:** Formulir pengajuan peminjaman dengan validasi interaktif.
- **Responsive UI:** Tampilan tabel dan menu yang rapi di Desktop maupun Mobile.
- **Realtime Filtering:** Pencarian dan filter data peminjaman secara instan.

## 🛠 Teknologi
- React + Vite
- TypeScript
- Tailwind CSS (v4)
- Axios (Integrasi API)
- JWT Decode (Manajemen Sesi)
